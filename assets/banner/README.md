# Banner assets

## `identity-map.svg`

The public profile hero combines the confirmed name and `Gamoventure1175` handle with
a backend-to-systems/data flow. Its `G / 1175` mark, component boundaries, feedback
path, and compact personal telemetry turn the profile identity into a system diagram
rather than a generic decorative header.

The 1280 × 360 SVG is original, dependency-free, and accessible. Its fixed dark
technical surface stays consistent in both GitHub themes and uses only SVG primitives,
system fonts, and a restrained blue signal color.

## `system-map.svg`

`system-map.svg` is an original, dependency-free profile banner built from geometric SVG primitives. Its 1280 × 320 canvas and restrained system-diagram motif are intended to remain legible at typical GitHub README widths.

The composition is a visual shorthand for the profile's engineering principles:
inputs enter from the left, pass through explicit component boundaries, fan out to
connected outputs, and return through a feedback path. It represents structure,
information flow, and deliberate iteration rather than a specific deployed system.

The palette follows the viewer's light or dark color preference. A full-surface
theme-aware background keeps the fine grid and module fills legible in either mode.
The artwork includes an accessible title and description; its internal geometry is
hidden from assistive technology to avoid noisy element-by-element announcements.
