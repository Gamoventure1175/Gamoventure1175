# Photos

`profile.jpg` is an original portrait supplied by Gaurav Abhiman Mahajan for this
profile. The source file was already JPEG-encoded and was renamed from a `.png`
extension so its filename matches its actual format. No visual edits were applied.

`profile-avatar.jpg` is the reusable square crop generated from that source by
`scripts/generate_profile_photo.py`. The crop preserves the photograph's natural warm
palette; it is not color-graded or AI-modified.

`profile-avatar-round.png` applies only an antialiased circular transparency mask to
the same crop. It is the version shown in the README; the photograph itself is not
retouched or regenerated.
