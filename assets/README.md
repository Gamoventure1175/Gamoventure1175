# Assets

Assets in this directory belong to the profile and should be original whenever
possible.

| Directory | Purpose | Publication rule |
| --- | --- | --- |
| `banner/` | Wide profile compositions | Must remain legible in light and dark themes |
| `icons/` | Small symbols and documented brand marks | Keep sets curated and provenance explicit |
| `illustrations/` | Diagrams and geometric studies | Must communicate a real idea |
| `photos/` | Portrait, workspace, and speedcube photography | Requires user-supplied originals |
| `signature/` | Monogram or converted handwritten mark | Requires approved initials or source material |

Do not add undocumented logos, stock art, remote tracking images, or assets with
unclear licensing. A small brand mark may be used as a truthful identifier when its
source and license are recorded. Document source material and transformation notes
beside every asset that is not created directly in this repository.
