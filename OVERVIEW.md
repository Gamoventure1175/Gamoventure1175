# Project Brief — GitHub Profile

## Goal

Design and build a GitHub profile that accurately represents me as a junior developer. This is **not** intended to be another version of my resume. It should communicate curiosity and personality without presenting every technical interest as expertise.

The result should feel timeless, understated, and highly curated. Every section should exist for a reason.

Avoid generic GitHub profile trends such as:

* Profile view counters
* Technology badge walls
* Animated contribution graphs
* Snake animations
* Random GIFs
* "Hi there 👋" introductions
* Decorative widgets with little informational value
* Generic motivational quotes

The overall feeling should be closer to an engineer's notebook than a social media profile.

# Visual Direction

The profile should communicate professionalism without feeling corporate.

Design inspiration should come from:

* engineering notebooks
* technical documentation
* terminal interfaces
* architecture diagrams
* technical books
* Swiss typography
* clean editorial layouts

The published visual system should be dark-first: near-black graphite, charcoal and
slate surfaces, cool off-white type, and restrained blue or cyan accents. Subtle
gradients, glass-like depth, and glow may be used only when they clarify hierarchy or
motion.

Avoid warm brown, coffee, or earthy theme treatments. Also avoid excessive colors,
unnecessary decoration, and visual clutter.

Whitespace and typography should do most of the work.
